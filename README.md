# Business AI Agent

A RAG-powered business planning agent built with LangChain, FastAPI, and ChromaDB.

## Features
- AI-generated business strategy
- Marketing plan and email campaign
- Task breakdown
- RAG support for custom document context
- Real-time streaming responses
